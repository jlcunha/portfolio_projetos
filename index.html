Projetos em Ciências de Dados:

Previsão de Vendas das proximas semanas, usando Machine Learning.

Neste projeto é feita a previsão de vendas das próximas 6 semanas da rede de farmácias Rossmann. Usando dados históricos das vendas para treinar os modelos de Machine Learning, e como forma de entrega um chatbot no Telegram.

Foram implementados 5 modelos diferentes e no final o modelo escolhido foi o XGBoost. Para colocar o modelo em produção foi utilizado flask e a cloud do Heroku, e então foi criado o chatbot no Telegram para que as previsões possam ser acessadas de qualquer lugar.

As ferramentas utilizadas foram:

  • Python, Pandas, Pyenv, Jupyter Notebook e PyCharm.
  • BorutaPy, NumPy, Scikit-Learn e SciPy.
  • Matplotlib e Seaborn.
  • Git, GitHub e Gitlab
  • Flask e Heroku Cloud e Telegram.

Coleta de Dados no Mercado de Moda.

Com o objetivo de ter mais informações sobre produtos e preços no mercado de moda, nesse projeto foi realizado Web Scraping( Coleta de Dados ) de um e-commerce de roupas.

Tratamos esses dados utilizando Regex e outras técnicas. E para a parte final do ETL, o Load foi criado um banco de dados em SQLite.

Utilizamos o Cron para automatizar a coleta de dados, e também foi feito um Dashboard em Panel e HVPlot para visualizar algumas informações.

  • Python, Pandas, Pyenv, Jupyter Notebook e PyCharm.
